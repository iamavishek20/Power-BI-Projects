Amazon Prime Subscription Analytics Dashboard (Power BI)

Overview:
This Power BI dashboard analyses Amazon Prime subscription behaviour, churn patterns, user engagement, and demographic insights. It was created as part of my MSc International Business with Data Analytics module.

Key Features:

Renewal trends (auto vs manual)

Monthly vs annual subscription analysis

Demographic segmentation (age, gender)

Feedback rating insights

Device-based engagement patterns

KPI visuals using DAX measures

Tools Used:
Power BI • DAX • Data Modelling • Data Cleaning • Visualization • Business Intelligence
